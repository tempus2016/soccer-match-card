# Soccer Match Card

Custom Home Assistant Lovelace card showing soccer match info and team logos.

## Installation

1. Copy `soccer-match-card.js` to `/config/www/` in Home Assistant.
2. Add it as a resource:
   - `/local/soccer-match-card.js`
   - Type: JavaScript Module
3. Add the card to your dashboard.
4. Edit the card and select your sensor entity.

## Features

- Pulls home/away teams and logos from TheSportsDB.
- Custom styling (Sky Sports vibe).
- Editable in Home Assistant UI.
